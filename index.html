Caesar Cipher

The Caesar Cipher is a simple substitution cipher where each letter in the plaintext is shifted a certain number of places down or up the alphabet. It is named after Julius Caesar, who used it in his private correspondence.

How It Works

To encrypt a message, you shift the letters by a fixed number of positions in the alphabet. For example, with a shift of 3:

  • A becomes D
  • B becomes E
  • C becomes F

To decrypt the message, you shift the letters back by the same number of positions.

Caesar Cipher in Ruby

Here is a simple implementation of the Caesar Cipher in Ruby: 

# Caesar Cipher implementation in Ruby
def caesar_cipher(text, shift)
    alphabet = ('a'..'z').to_a
    ciphered_text = text.chars.map do |char|
        if alphabet.include?(char.downcase)
            shifted_index = (alphabet.index(char.downcase) + shift) % alphabet.size
            char == char.upcase ? alphabet[shifted_index].upcase : alphabet[shifted_index]
        else
            char
        end
    end
    ciphered_text.join
end

# Example usage
plaintext = "Hello, World!"
shift = 3
encrypted_text = caesar_cipher(plaintext, shift)
puts "Encrypted: #{encrypted_text}" # Output: "Khoor, Zruog!"

In this example, the caesar_cipher method takes a string and a shift value. It then encrypts the string by shifting each letter by the specified number of positions. Non-alphabetic characters remain unchanged.

Decryption Example

To decrypt the text, you simply call the same method with a negative shift value: 

# Decryption example
encrypted_text = "Khoor, Zruog!"
shift = -3
decrypted_text = caesar_cipher(encrypted_text, shift)
puts "Decrypted: #{decrypted_text}" # Output: "Hello, World!"
